## Реалізація програми мовою Common Lisp (текст програми).
```lisp
 (defun F (i)
  (labels ((df (x) (coerce x 'double-float)))
    (cond
      ((= i 1) (df 1))
      ((= i 16) (df 2))
      ((and (>= i 2) (<= i 15))
       (let ((prev (F (1- i))))
         (+ (* (df 2) prev) (* (df 5) (sin (df i))))))
      ((and (>= i 17) (<= i 30))
       (let ((prev (F (1- i))))
         (+ (/ prev (df 2)) (* (df 5) (cos (df i))))))
      (t (error "index ~A out of range" i)))))
```
